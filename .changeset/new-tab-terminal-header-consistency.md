Make the new-tab empty state consistent when the docked terminal is open. Opening a terminal from an empty tab used to collapse the whole create surface to just the mascot peering down at the terminal. It now keeps the same left-aligned header as the normal empty state — the blob beside the "Create something great." headline and its subtitle (or the onboarding copy for a brand-new project) — while still omitting the composer and starter-pack list so they don't crowd the terminal below.
